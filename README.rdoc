Sample Adhearsion application that runs alongside a Rails app 
for the provisioning of routes. 

Dialplan is currently incomplete.

To get started:

rvm use 1.9.2
rvm gemset create callengine
bundle install 
bundle exec rails s
bundle exec ahn -

Read the following post to point a Tropo number to your app:

http://libryder.com/2011/using-tropo-to-connect-inbound-calls-to-adhearsion
